Stellar Slack bot

This is a Slack Bot for the Stellar Network.

It works in two ways: - Streaming: It streams all operations on the stellar network to a channel - Queries: It allows users to query the stellar network for basic information such as accounts details, balance and transaction details.

Usage

You can query the Stellar network for account information and transactions as follows: /stellar account [account_id] /stellar balance [account_id] /stellar tx [transaction_id]
